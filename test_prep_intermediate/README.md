Question 1: Loops

Write a Python function find_primes(n) that takes an integer n and returns a list of all prime numbers from 2 up to n (inclusive).

Requirements:

Use loops (for or while).

Avoid using external libraries like sympy.

Optimize it a bit to reduce unnecessary checks (e.g., check divisibility only up to the square root of the number).

Example:

find_primes(10)  # Output: [2, 3, 5, 7]
find_primes(20)  # Output: [2, 3, 5, 7, 11, 13, 17, 19]
